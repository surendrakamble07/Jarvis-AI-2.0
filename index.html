GET STARTED

Install in Minutes

Follow these steps and JARVIS will be running on your machine in under five minutes.

01

Download the ZIP

Click the Download button above or go to the Downloads section. Save JarvisAI.zip anywhere on your PC.

02

Extract the Files

Right-click JarvisAI.zip → Extract All. Choose a destination and extract the contents to a clean folder.

03

Install Requirements

Open a terminal inside the folder and run: pip install -r requirements.txt. Python 3.8+ required.

04

Run main.py

Execute python main.py in the terminal. JARVIS will initialise and greet you.

05

Enjoy JARVIS

Say "Hey JARVIS" and start commanding your AI. Welcome to the future.

FAQ

Common Questions

Everything you need to know before getting started.

JARVIS AI requires Windows 10 or 11, Python 3.8 or higher, a working microphone, and an active internet connection for AI features. Minimum 4 GB RAM recommended.

Python 3.8, 3.9, 3.10, or 3.11 all work. Python 3.12 is not fully tested. Download Python from python.org and make sure to check "Add Python to PATH" during installation.

Open the config.json file inside the extracted folder and paste your OpenAI API key (and optionally other keys). Full instructions are in the Installation Guide PDF.

Check that your microphone is set as the default recording device in Windows Sound Settings. Make sure you've installed pyaudio correctly. Speak clearly, about 30 cm from the mic.

Yes. JARVIS has a modular command system. You can add Python functions to the commands/ directory and register them in commands.json. See the guide for details.

JARVIS AI itself is completely free and open-source. You will need your own API keys for OpenAI and other AI services, which may have usage costs depending on your provider's pricing.
